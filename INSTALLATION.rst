============
Installation
============

At the command line::

    $ pip install wikiseriesarjancli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiseriesarjancli
    $ pip install wikiseriesarjancli

Or, if you are using pipenv::

    $ pipenv install wikiseriesarjancli
Or, if you are using pipx::

    $ pipx install wikiseriesarjancli
